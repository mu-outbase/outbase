# OUTBASE Record-01 スクショ検証メモ

作成日：2026-07-11  
ベース：OUTBASE_ROUTE08_UI_REBUILD_CLEAN_v6_基準寄せ承認版  
対象：記録画面の項目・操作・小型シート

## 反映内容

- CLEAN v6の色、カード形状、余白、下部ナビを維持
- 記録先を「コタ通常散歩」として常時表示し、タップ変更を追加
- 状態表示を「GPS取得中 / オフライン保存 / 画面ON」に整理
- GPS値を「GPS点 / 距離 / 経過時間 / 記録数」に変更
- 記録中の主操作を「一時停止」、終了を独立した小型ボタンに変更
- クイック記録を「話す / 撮る / 動画 / 場所 / ピン / メモ」に統一
- メモ入力、終了確認、記録詳細、記録先変更を小型シートで実装
- 地図上の写真・メモ・現在地点をタップできる記録点として追加
- 432×768の一画面内に収まることを確認

## 動作検証

実ブラウザで以下を確認。

- ピン保存で記録数が18→19へ増加：合格
- メモ入力・保存で記録数が19→20へ増加：合格
- 一時停止→再開：合格
- 終了確認シートの開閉：合格
- 記録先を「未確認箱へ仮保存」へ変更：合格
- 地図上の記録点から記録詳細シートを表示：合格
- 画面スクロール高768px、一画面表示：合格
- `node --check src/app.js`：合格
