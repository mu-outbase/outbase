# OUTBASE PREP IMPLEMENTATION LOCK v1.2 / Prep-01.2

作成日：2026-07-11

## 準備ホーム
- 主役予定あり：予定別の天気／ギア／料理／買物／ルートを表示。
- 主役予定なし：同じレイアウトで「準備ノート」を表示。
- 予定なし時も下部ナビまで432×768の一画面。
- 補助操作は深緑の一体バーへ整理。

## 共通準備ノート
- 保存キー：`outbase_prep_common_v1`。
- moduleごとに `items` と `note` を保持。
- 対象：天気、料理、買物、ルート。
- 料理案は複数保存可能。
- 主役予定がある場合、保存案を予定別の `customItems` へコピー可能。
- 共通ノート側の元データはコピー後も保持する。

## ギア
- 共通ギア管理：`outbase_gear_library_v1`。
- 予定別持出し：`outbase_prep_v1` 内の `gearIds`。
- 主役予定なしでも所有ギア台帳を編集可能。

## モーダル誤タップ防止
- シート中は `.bottomNav` を無効化。
- captureフェーズで下部ナビのpointer/clickを遮断。
- シート終了時に `#outbaseNavShield` を下部96pxへ設置。
- シールドは終了後約720msで解除。
- 外側タップはpointerdown時点からナビ遮断を開始する。

## 見た目
- CLEAN v6のクリーム背景、深緑カード、暗い真鍮ゴールドを維持。
- 白い2枚カードの補助バーを廃止。
- 予定なし画面も選択中画面と同じ密度・階層で表示。
- 詳細シートのチェック、保存案、入力、メモをカード状に整理。

## 維持LOCK
- Plan-01.2、Record-02.2、Prep-01.1。
- 固定順：天気、ギア、料理、買物、ルート。
- 駐車場はルート内。
- 持ち物カードなし。
