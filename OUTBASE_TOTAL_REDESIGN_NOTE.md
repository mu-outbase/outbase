# OUTBASE total redesign

## 今回の目的
- カレンダーだけではなく、全画面のデザイン言語を統一。
- テスト画面っぽいラベル感・カード感を減らす。
- Snow Peak / Louis Poulsen / MasterWal を意識した、静かで上質なトーンに寄せる。
- 予定・準備・現地記録・思い出の導線を整理する。

## 主な変更
- 全画面の余白・色・タイポ・面構成を統一。
- ホームを「入口整理」型に再構成。
- カレンダーは全体俯瞰、下に日別詳細の2段構成。
- 準備は料理 / 買い物 / ギア / 天気の流れに統一。
- ギアは一覧 / ボックス / 積込 / 乾燥修理 / 取込候補を用意。
- 現地記録は散歩・GPS・スポット・犬友達・非公開体調メモを統合。
- 思い出はメモ・場所・犬友達・バックアップを整理。
