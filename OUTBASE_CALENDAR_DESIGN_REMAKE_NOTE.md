# OUTBASE calendar + design remake

## 反映内容
- カレンダーの単体予定と連日予定を見た目で分離。
- 単体予定は小チップ、連日予定は横につながるバー。
- 繰り返し予定は ↻ 表示。
- 日付未設定はカレンダー本体に混ぜず別枠。
- 左右スライド月移動、タップ選択、ダブルタップ追加。
- テスト画面っぽいラベル/カード感を抑制。
- Snow Peak / Louis Poulsen / MasterWal の方向性として、生成り・暖色光・深緑・木・マットな余白に寄せた。
- ギア登録/変更、ボックス管理、積込、乾燥/修理、取込候補。
- 散歩は通常/キャンプ場、地図、GPS、スポット、犬友達、排泄ログは非公開の体調メモ内。
