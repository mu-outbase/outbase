# OUTBASE_BOOT v9.5 / Core06.10 カレンダー主役復帰反映

## 位置づけ
E1 / E2 / E3 / F1 は正本準拠の完成版として扱わない。
理由は、カレンダー主役の製品体験から外れ、テスト画面・モード一覧・ボタン羅列へ戻ったため。

本版は、OUTBASEを「カレンダー中心の使うアプリ」に戻す復帰版。

## LOCK
- 起動直後は予定タブ。
- 予定タブの主役は月間カレンダー。
- 次のキャンプ予定、準備状況、未確認件数、当日運転席への導線を上段に置く。
- 下ナビは「予定 / 探す / 準備 / ＋ / 思い出」。
- 全モードは画面に羅列しない。
- モードの中身は予定・準備・当日運転席・＋・思い出の中に埋め込む。
- テスト画面に見えるUIは禁止。
- 予定・記録・写真・メモ・GPS・コタ情報はユーザー操作なしに勝手に上書き/統合/削除しない。

## 実装方針
### 予定
月間カレンダー、次のキャンプ、複数予定、予約情報、準備/当日導線。

### 探す
犬可、ドッグフリー、温水、景色、距離、季節条件、候補比較、予定化。

### 準備
買い物、料理、ギア、コタ、ルート、天気、積み込み、出発前タスク。

### 当日運転席
予定から開く文脈画面。
出発前、往路、到着受付、レイアウト、設営、滞在、料理、就寝/翌朝、撤収、帰路、帰宅後。

### ＋
写真、動画、声、メモ、GPS、あとで、活動開始。
活動は散歩、ドライブ、設営、料理、撤収、キャンプ場散歩、場内探索。

### 思い出
時系列、未確認箱、レビュー、反省、次回改善、Google Photos候補、復旧。
