# OUTBASE BOOT v6.08 / Core06.08 人間中心UX整理反映

## 位置づけ
Core06.08は、Core06.07で残った「使いにくさ」「見にくさ」「迷子感」を、人間工学・認知負荷・行動導線を優先して整理した版。

## 反映内容
- 全ページ下部の余白を圧縮し、固定ナビに必要な最低限だけ残す。
- 下ナビを黒背景から明るい面＋濃色文字へ変更し、視認性を改善。
- 記録中の下バー「＋」を「戻る」状態に変え、別ページから記録へ戻る導線を明確化。
- 記録中インジケーターは情報表示のみ。準備・探す・当日・思い出閲覧を邪魔しない。
- カレンダーは数字だけでなく日付セル全体をタップ対象化。
- カレンダー日付数字は左上固定。
- 記録開始画面は「何をする？」の優先3択＋場内散歩へ整理。
- 散歩地図はGPS未取得でも安定した地図パネルを表示し、現在地取得でルート開始の文脈に変更。
- 記録コックピットはファーストビューで「今のモード・時間・距離・切替・よく使うボタン・地図」が分かるよう整理。

## UX原則
1. 探させない。
2. 迷わせない。
3. 押す場所を広くする。
4. 今の状態を一目で出す。
5. 他ページ閲覧中に記録へ強制的に戻さない。
6. 見た目より先に行動導線を優先する。
