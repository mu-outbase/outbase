# OUTBASE Plan Implementation LOCK v1.1 / Plan-01.1

## 日付・月操作
- 日付1回目：選択。
- 選択中の同じ日を2回目にタップ：その日付で新規追加。
- 異なる日をタップ：選択日の移動のみ。
- 左右スワイプ：前月／翌月。

## 追加画面
- 外側タップで閉じる。
- 下方向スワイプで閉じる。
- 未入力なら即時閉じる。
- 入力途中は破棄確認。
- 開始・終了の日付と時間、終日を設定。

## 複数・繰り返し
- 同日複数予定を別IDで保持。
- 毎日／毎週／毎月／毎年。
- 間隔、曜日、終了日、回数を設定。
- 繰り返し予定はseriesIdで関連付ける。

## 状態
- 日時から予定／実施中／終了済みを自動判定。
- 手動の「完了にする」は削除。
- 例外状態は今後の詳細メニューで扱う。

## 種類
- 初期：散歩／ドライブ散歩／キャンプ／ショッピング／イベント／その他。
- 未選択：汎用「予定」。
- 追加／名称変更／色変更／削除。
- 削除後も過去予定の種類名は保持。

## 同行者
- 選択式・複数選択。
- ひとりは排他。
- 初期：ひとり／リン／コタ。
- 追加／名称変更／削除。
- 削除後も過去予定の同行者名は保持。
