# OUTBASE BOOT v8.1 / Core08-B ホーム司令塔

## 反映内容
- Homeを単なるカレンダーではなく、OUTBASEの司令塔にする。
- 既存カレンダー情報は自動修正・上書きしない。
- カレンダーの上に「次のキャンプ」「今日の予定」「記録中」「データ保護状態」を集約表示する。
- 準備・当日・記録・探すへの入口をHome上部に置く。
- カレンダー本体と予定編集機能は維持する。

## LOCK継承
- Core08-A3データ保護基盤を継承。
- 自動修正、自動統合、自動削除、自動上書きは禁止。
- Home司令塔は表示と導線のみ。データ本体は勝手に触らない。

## 確認項目
- Home上部に司令塔が出る。
- カレンダーは下に残る。
- 予定名が鹿沼/赤城山などに勝手に変わらない。
- 準備へ押下で準備トップへ遷移する。
- 記録中なら記録に戻る導線が出る。
