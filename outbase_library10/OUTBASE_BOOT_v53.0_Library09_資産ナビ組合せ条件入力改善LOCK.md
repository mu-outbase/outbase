# OUTBASE BOOT v53.0

## Library-09 資産ナビ・組合せ・条件入力改善 LOCK

- メイン正本は MASTER_v142 を維持する。
- 実装ベースは Library-08。
- 下部5タブを資産画面表示中も維持する。
- 資産上段・ギア内タブを小型固定表示とし、タブ切替時のスクロール位置引継ぎを廃止する。
- カテゴリ絞込みは濃い文字・明確な選択状態とする。
- 「つながり」は、必ず一緒／条件で一緒／取付・装着／対応／燃料／収納先へ簡略化する。
- 関係先はギアだけでなく、車・自転車、保管場所も選択可能とする。
- 条件付き関係に、インナー使用時・寒い時・厳寒時・雨天・電源なし等を保存できる。
- タグは候補選択＋自由入力とする。
- 名称からカテゴリ・役割・代表タグを補完する。
- 購入金額×数量の小計、資産全体の合計を表示する。
- セットは複数所属を許可し、各構成品を必須／任意に分ける。
- セットに季節・気温・電源・天候等の使用条件を保存する。

FIELD03正ベース／RESTORE04.8ルート系／CLEAN v6／Plan01.2／Record02.2／Prep01.2／Library08までの資産機能。
