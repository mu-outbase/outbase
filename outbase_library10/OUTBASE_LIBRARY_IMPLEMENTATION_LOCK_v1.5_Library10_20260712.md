# OUTBASE Library Implementation LOCK v1.5

## 採用内容

1. 取込画面
   - 大型ヒーロー依存をやめ、4つの目的カードを先に表示。
   - まとめて登録、Excel、PDF・写真、バックアップへ直接移動。

2. 保管場所
   - 保管場所カードを展開すると、その場所に登録されたギア・車両を表示。
   - 中身から直接対象の編集画面へ移動。

3. ペット台帳
   - 誕生日、狂犬病次回、混合ワクチン次回、健康診断次回、病院・登録番号を保持。
   - カード上で期限概要と履歴件数を表示。

4. 車両台帳
   - 車検期限、保険期限、次回整備、走行距離を保持。
   - 装着・関連ギアを車両カードから確認。

5. 共通ポップアップ操作
   - 背景タップで閉じる。
   - 下方向へ一定量スワイプで閉じる。
