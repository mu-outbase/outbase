# OUTBASE Record-02 実GPS・大地図・位置自動補完・駐車位置 実装LOCK

## 正ベース
- CLEAN v6 UIデザイン正式承認版を継承
- Record-01の記録先分離・セッション操作・小型シートを継承
- FIX1〜FIX10、旧REBUILD17/18/18.1、RESTORE01は不使用

## 記録画面
- 上部チップ：表示中プランの確認・切替
- 細い保存先バー：写真・動画・音声・メモ・場所を保存する先
- 重複していた「表示／保存」バーは削除
- 記録カードは記録状態と同伴者に限定
- 地図を一画面内で取れる最大領域へ拡大
- 表示数値は「距離」「経過時間」のみ
- GPS点数・記録数は内部管理とし通常画面には表示しない

## 実GPS・地図
- Geolocation `watchPosition` により現在位置を追跡
- OpenStreetMapタイル上へ現在地・軌跡・保存ピンを表示
- 「現在地」：地図を現在位置へ戻す
- 「場所を記録」：その時点の場所を即保存
- Google Map：現在位置をGoogle Mapsで開く
- GPS記録は一時停止中に止め、再開時に再開

## クイック記録
- 話す／撮る／動画／メモの4入口
- 全記録に日時・保存先・散歩セッション・位置を自動添付
- 写真・動画は撮影開始時点、音声は話し始め、メモは保存時点を基準にする

## 音声文字起こし
- 「話す」を約0.3秒以上長押しして開始
- 押している間は地図上に認識中の文字をリアルタイム表示
- 指を離すと自動保存
- 保存直後に「修正」「取消」を表示
- 対応ブラウザでは Web Speech API を使用

## 位置の自動補完
- GPSが瞬間的に取れなくても記録を止めない
- 記録時刻と前後のGPS軌跡から最も近い位置を自動で紐付ける
- 新しいGPS点を取得するたび、位置未確定の写真・動画・音声・メモ・ピンを自動補完
- ユーザーへ「位置確認待ち」を一件ずつ確認させない
- 散歩全体で位置を一度も取得できなかった場合だけ、終了時に一度通知する

## 場所ピン
- 水飲み場／トイレ／駐車場／自販機／ベンチ／日陰・休憩／出入口／注意場所／その他
- 種類未選択でも未分類ピンとして保存可能
- 場所メモ、写真、音声を同じ地点に紐付け可能

## 駐車位置
- 駐車場ピンはGPS位置・時刻を即保存
- 階・色・柱・区画番号は長押し音声から自動整理
- 案内板・柱・区画写真はOCRで読み取り、音声と突き合わせて補完
- 手入力は必要時だけ開く補助欄
- 駐車位置保存後は、記録画面の地図上に「駐車位置」を常時表示
- 他タブでも「車に戻る」チップから1タップで呼び出せる
- 駐車位置カードから、地図確認／徒歩ルート／写真／音声・メモを開ける
- 「車に戻った」で常時表示を終了する
- 駐車情報はlocalStorage・IndexedDBへ保存し、画面移動や再読込後も復元する

## 外部機能・実機条件
- GPSはHTTPS環境と位置情報許可が必要
- OpenStreetMapの地図タイル表示は通信環境に依存
- 音声文字起こし精度は端末・ブラウザに依存
- 写真OCRはTextDetector、利用不可時はTesseract.jsを使用するため初回通信が必要な場合がある
