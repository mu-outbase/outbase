# OUTBASE RECORD IMPLEMENTATION LOCK v2.1
## Record-02.1 開始状態・距離・時間・GPS表示同期

- `idle` を記録画面の正式な初期状態とする。
- `idle` 中は現在地取得を許可するが、時間・距離・軌跡へ加算しない。
- `active` 中のみ `watchPosition` の点を今回セッションへ追加する。
- `paused` 中はwatchPositionと時間加算を止める。
- 状態表示と主ボタンは同一stateから描画し、別管理しない。
- 終了時は完了セッションの時間・距離・軌跡を履歴データへ保存してから、記録画面の値を0へ戻す。
- 未解除の駐車位置は記録セッションと別ライフサイクルで保持する。
- Record-02のUI骨格・大地図・クイック記録・駐車位置機能は変更しない。
