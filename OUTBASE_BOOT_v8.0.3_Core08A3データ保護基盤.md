# OUTBASE BOOT v8.0.3 Core08-A3 データ保護基盤

## 目的
カレンダー予定・記録済み情報・写真・メモ・GPS・料理・ギア・コタ情報を、ユーザー操作なしに勝手に修正・統合・上書き・削除しないための保護基盤を追加する。

## LOCKルール
- 予定ID・記録IDを保護する
- 予定名・日付・種別は別情報で勝手に上書きしない
- 同じ名前/似た名前でも勝手に統合しない
- 削除や一覧から外れた情報は復元箱へ保護する
- state変更前にバックアップを残す
- 変更履歴・保護ログ・競合ログを残す

## 今回の直接対応
鹿沼と赤城山のような別予定を、準備保存時に nextProject へ上書きしない。カレンダー予定を選択している場合は、そのカレンダー予定だけに準備情報を保存する。
