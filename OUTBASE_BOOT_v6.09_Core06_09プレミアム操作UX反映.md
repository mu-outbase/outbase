# OUTBASE BOOT v6.09 / Core06.09 プレミアム操作UX反映

## 位置づけ
Core06.08の人間中心UX整理を正本として、触った時の気持ちよさ・迷子防止・実地図・MVP機能の置き場を反映した版。

## 反映内容
- View Transition API による画面遷移の滑らか化
- ボタン押下時のマイクロインタラクション追加
- 散歩マップを Leaflet / OpenStreetMap 表示に対応
- GPS未取得時も地図枠と取得導線を表示
- GPS watch が他ページを強制的に記録画面へ戻す挙動を停止
- 記録終了・破棄後、10分以内の復旧導線を追加
- 探す画面の大型入力を圧縮し、条件をコンパクト化
- 準備ページにMVP機能の置き場を追加
  - 詳細な買物リスト
  - 朝昼晩の料理計画
  - ギア管理
  - ルート/経由地
- 下バーと記録中インジケーターの視認性を継続改善

## 重要ルール
- 記録中でもカレンダー、探す、準備、当日、思い出へ自由に移動できる。
- GPS更新だけで画面を強制的に記録へ戻さない。
- 記録中の復帰導線は下バー中央と記録中インジケーターで示す。
- 誤終了・誤破棄は10分以内に記録画面から復旧できる。
- MVP機能は準備ページを中心に段階的に展開する。
