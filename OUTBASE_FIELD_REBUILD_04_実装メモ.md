# OUTBASE FIELD REBUILD 04 実装メモ

## 散歩
- うんち / おしっこ / 体調 / 危険 は削除。
- 散歩中に残すのは、ピン / 写真 / 動画 / 長押し音声メモ。
- 音声は対応端末でリアルタイム文字起こし。
- 開始 / 休憩 / 再開 / 終了は残す。
- コタ散歩と場内散歩は、細かい健康管理ではなく「軌跡 + ピン + メディア」へ寄せた。

## ドライブ
- ドライブ中の記録は、ピンと長押し音声メモだけ。
- 操作パネルは通常OFF。
- 操作パネルを開いて、ボタン操作後に自動OFF。
- 自動OFF秒数は 1〜10秒でカスタム可能。初期値は3秒。
- Google Mapsはナビ、OUTBASEは記録係。

## 出発/到着
- 自宅出発固定をやめた。
- 文脈により、散歩出発/帰宅、出発/到着、自宅出発/帰宅到着を表示。
- 親セッション開始で勝手に移動モードへ切り替えない。
