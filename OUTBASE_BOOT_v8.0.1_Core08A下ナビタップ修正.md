# OUTBASE BOOT v8.0.1 Core08-A 下ナビタップ修正

## 反映内容
- 下段バーの各項目をタップしても予定/カレンダーに戻される問題を修正。
- bottom-nav は個別ボタン直付けではなく、`.bottom-nav` にイベント委譲して `closest('[data-route]')` で押下項目を確定する。
- nav ボタンに `type="button"` を明示する。
- 起動時は保存済み `currentRoute` を優先し、強制的に home へ戻さない。
- `src/core/router.js` をCore08系のキャッシュバスターに更新。

## 確認ポイント
- 下段バーの「探す」「準備」「＋」「当日」「思い出」を押して、それぞれ該当ページへ移動する。
- タップ直後に予定/カレンダーへ戻されない。
- ワークスペース表示中は下ナビが隠れるCore08-A方針を維持する。
