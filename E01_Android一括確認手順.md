# E-01 Android一括確認手順

## 重要
不具合1個ずつではなく、以下を1回の確認回で一巡する。
異常があった画面名と「戻った先」だけ記録する。

## A. 基本5タブ
予定 → 探す → 準備 → 記録 → 思い出
- 各タブへ移動できる
- 実行中Activityがある場合も消えない

## B. ネスト画面
1. 準備メモ一覧 → 新規追加
   - 戻る1回: 準備メモ一覧
   - 戻る2回目: 準備画面
2. Chappy相談 → 保存先
   - 戻る1回: Chappy相談
   - 戻る2回目: 元画面
3. 思い出一覧 → 詳細 → レビュー
   - 戻る1回: 思い出詳細
   - 戻る2回目: 思い出一覧
4. 予定 → プラン詳細 → 編集
   - 戻る1回: プラン詳細
   - 戻る2回目: 予定

## C. 開始地点
- 予定から記録
- 準備から記録
- 予定なしで今すぐ記録
- 実行中Session再開
- 思い出から過去整理
すべてで戻る先が開始元の1階層下になること。

## D. 複数同時進行
- プランAを準備中
- プランBを記録中
- プランCの思い出を開く
戻る操作でプランやActivityが勝手に切り替わらないこと。

## E. 終了条件
- アプリ終了・ブラウザ外遷移: 0
- 予定ホームへの意図しない復帰: 0
- 2階層以上を一度に閉じる: 0
- 閉じるべき画面が残る: 0
