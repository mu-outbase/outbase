# OUTBASE Library 実装LOCK v1.1

## 採用UI
1. 資産は全画面の独立ページ。
2. 白系ベース＋深緑アクセント。
3. 上段: ギア／車・自転車／ペット／保管場所／取込・出力。
4. ギア選択時の下段: 概要／アイテム／セット／カスタム／在庫／履歴／つながり。
5. 上下2段のタブは常時同じ位置に固定。
6. 各タブのスクロール位置を個別保持。
7. 本文・入力・ボタン16px基準。小さな英字ラベルも12px以上。
8. ポップアップは登録・編集など短時間操作に限定。

## 維持機能
資産ダッシュボード、大量ギア検索、セット、関係、カスタム構成、消耗品・燃料、履歴、車・自転車、ペット、保管場所、JSON入出力。
