# OUTBASE RESTORE04.18 FIELD03正ベース ルート以外全体UIUX再統合 メモ

## 位置づけ
- ルート画面は04.8系の一画面性を承認済みとして保護。
- ルート以外は未承認として、トップ/予定/探す/準備/記録/思い出/デザイン/スクロールをまとめて再統合。
- 04.13 Context Model と 04.15 外出属性モデルは内部に維持。
- 04.17へカレンダーだけを追記する小修正ではなく、未承認範囲の全体UIUX再統合版。

## 実装内容
- 予定トップをカレンダー主役に再整理。
- 今日 / 次の外出 / 選択日の予定 / 未確認記録をトップ側に配置。
- 外出タイプ選択は「＋新規予定の種類を選ぶ」details内へ退避。
- 土曜/日曜/祝日の表示を追加。色だけでなく「土」「日」「祝」と祝日名で表示。
- 連泊/連動予定をカレンダー内のバー表示に変更。
- 日付1回タップで選択日表示、同日短時間2回タップで予定追加。
- 同一タブ内操作ではスクロール位置を保持。タブ切替時のみ先頭へ。
- 記録タブに散歩マップを主役級で配置。
- 散歩マップから散歩開始 / 現在地 / ピン / Google Mapへ進める。
- 準備/探す/思い出もルート画面寄りの質感に再統一。

## 壊していない対象
- 04.8系ルート出発逆算。
- Context Model。
- 外出属性モデル。
- 現地記録 / GPS / ピン / 写真 / 動画 / 音声系の既存関数。
