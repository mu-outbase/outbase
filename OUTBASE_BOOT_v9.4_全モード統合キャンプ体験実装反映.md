# OUTBASE BOOT v9.4 / Core08-F1 全モード統合キャンプ体験実装反映

作成日：2026-07-05  
扱い：Core08-F0 全モード中身統合LOCKを実装へ反映した版。E1/E2/E3は完成扱いにしない。

## 1. 実装LOCK

OUTBASEは、モード名ボタン集ではなく、キャンプ1回の流れを動かす。

- 予定 → 準備 → 当日運転席 → 各モード記録 → 思い出整理 → 次回改善 を一本化
- 全47モードをデータとして保持
- 各モードに、開始条件 / 初期表示 / 主要ボタン / 自動記録 / 連携 / 完了 / 思い出 / 次回改善 / UIルール / 保護ルール を持たせる
- 推定は未確認箱。勝手に確定しない
- 削除ではなく復旧・未確認・あとで・違うで保護する
- キャンプ中は整理させない。＋で写真/動画/声/メモ/GPS/あとでを残す

## 2. F1で実装した画面

- 予定：複数予定、予約情報取込、準備対象切替、MVP23確認
- 探す：犬可/温水/景色/距離/季節条件の候補カード
- 準備：買物/料理/ギア/ルート/コタ/天気を予定ごとに保持
- 当日：出発前→往路→到着→サイト→レイアウト→設営→滞在→夜/翌朝→撤収→帰路→帰宅後→思い出→次回改善の運転席
- ＋：常時フローティング。写真/動画/声/メモ/GPS/あとで、活動開始
- 思い出：未確認箱、時系列、次回改善、復旧/修正、Google Photos、オフライン、テストリセット
