## 0. 結論

04.15は「キャンプ / 散歩 / イベント」などの固定分類だけで画面を出し分けない。  
外出プランに属性を持たせ、属性から必要な入口だけを出す。

```text
OUTBASE = 外出OS
外出プラン + 属性 + セッション + 記録 + 関連リンクで吸収する
```

画面はシンプルにする。

```text
予定 / 探す / 準備 / 記録 / 思い出
```

内部は柔軟にする。

```text
Plan
├ templateType
├ attributes
├ sessions[]
├ records[]
└ links[]
```

## 3. 入口表示の決め方

画面に出す入口は、外出タイプ名だけではなく属性で決める。

```text
入口 = templateType + transport + stayType + petAccompanied + routeImportance + recordPurpose + status
```

例:

```text
ペット関連イベント / 同伴なし / 車 / チケットあり
→ ルート / 駐車場 / 入場時間 / ブース / 買い物 / 写真メモ / 次回同伴可否
```

```text
ドライブ散歩 / コタ同伴 / 車→徒歩 / 日帰り
→ ルート / 駐車場 / 現地散歩開始 / 水 / 休憩 / 写真 / ピン
```

```text
キャンプ / コタ同伴 / 宿泊あり / 車
→ ルート / 買い物 / 料理 / ギア / 天気 / 設営 / 場内散歩 / 撤収
```

## 4. 共通ヘッダー

ヘッダーは軽くする。  
Context Modelの説明は出さない。

```text
OUTBASE
表示中：赤城山キャンプ　⌄
保存先：コタ場内散歩　記録中
```

### ルール

- 表示中 = 今見ている外出プラン
- 保存先 = 今記録が入るセッションまたはプラン
- 表示中を変えても保存先は勝手に変えない
- 保存先が不明な時だけ未確認箱
- 1〜2行以内
- 小さく、邪魔しない

## 5. 下バー

下バーは固定。

```text
予定　探す　準備　記録　思い出
```

### タブの役割

```text
予定   = 外出を選ぶ / 作る / 切り替える
探す   = 次の候補を探す / 保存する
準備   = 表示中の外出に必要な準備をする
記録   = 今の行動を保存する
思い出 = 記録を見返す / 整理 / 改善する
```

## 6. ホーム

ホームは説明ではなく、最短入口。

```text
今日なにする？

[ 記録する ]
話す・撮る・場所・メモ

[ 準備する ]
ルート・持ち物・買い物・天気

[ 予定を見る ]
次の外出・候補・今日

小ステータス
表示中：赤城山キャンプ
保存先：コタ場内散歩
```

### 合格条件

- 3秒で押す場所が分かる
- 説明を読まなくても使える
- 記録入口が一番押しやすい
- 04.14のような説明カード化をしない

## 7. 予定タブ

予定タブは外出プランの入口。

```text
予定

次の外出
[ 赤城山キャンプ ]      7/18-7/20 / キャンプ
[ コタ通常散歩 ]        今日 / 散歩
[ 手賀沼ドライブ散歩 ]  候補 / 車→散歩
[ ペットイベント ]      人だけ / 物販・情報収集

[ 外出を追加 ] [ 今日 ] [ カレンダー ]
```

### 外出追加の最小フロー

```text
何をする？
[ 散歩 ] [ 車で散歩 ] [ キャンプ ] [ イベント ] [ 日帰り ] [ 下見 ] [ 買い物 ]

誰と行く？
[ 人だけ ] [ コタ ] [ 複数ペット ] [ 未定 ]
```

### 補足

- この2段階は予定作成時だけ
- 毎回選ばせない
- あとから変更できる
- ペットイベントでも「人だけ」を選べる

## 11. 記録タブ

記録は全タイプ共通でシンプル。

```text
記録
保存先：コタ通常散歩 / 赤城山キャンプ / 未確認箱

[ 話す ] [ 撮る ] [ 動画 ]
[ 場所 ] [ ピン ] [ メモ ]

[ セッション開始 ]
```

### セッション開始候補

```text
通常散歩:
  散歩開始

ドライブ散歩:
  往路 / 現地散歩 / 休憩 / 復路

キャンプ:
  設営 / 料理 / 場内散歩 / 撤収 / 片付け

イベント同伴なし:
  ブースメモ / 買い物メモ / 気になる商品 / 次回同伴可否

イベント同伴あり:
  会場内行動 / 休憩 / 買い物 / 犬友達 / 体調メモ

下見:
  駐車場 / トイレ / 犬OK / 景色 / 評価

買い物:
  購入品 / 気になる商品 / 在庫 / 価格

帰宅後整理:
  片付け / ギア乾燥 / 補充 / レビュー / 投稿整理
```

### 保存ルール

```text
明示された記録中セッションに保存
なければ表示中プランへ仮紐付け
それも不明なら未確認箱
表示中を変えても保存先は変えない
```
