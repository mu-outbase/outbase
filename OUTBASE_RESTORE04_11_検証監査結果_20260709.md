# OUTBASE RESTORE04.11 検証監査結果

## 結論
RESTORE04.11は、RESTORE04.8のルートAPIなし版を正ベースとして維持し、04.9/04.10の未承認積み増しを正ベースにせず、入口整理・プラン文脈・保存先保護だけを再実装した。

## 合格条件
- 上バーに「準備前 / 準備中 / 現地 / 帰宅後」を出さない
- 下バーは「予定 / 探す / 準備 / 記録 / 思い出」に固定
- カード支払い確認を通常導線から出さない
- キャンプ選択中だけルートAPIなし画面を出す
- 散歩選択中は、散歩記録・GPS・写真・音声・メモに集中する
- 散歩選択中にキャンプ用ルート文脈を表示しない
- プランチップは機能入口ではなく、主役プラン確認/切替に限定する
- 記録は主役プランへ紐付け、不明時のみ未確認箱へ逃がす
