# OUTBASE PLAN IMPLEMENTATION LOCK v1.2 / Plan-01.2

作成日：2026-07-11

## 実装対象
- 日付マスと予定バーのタップ分離
- 同一日2回目タップのみ新規追加
- 日付マス上からの月左右スワイプ
- 全予定系シートの外側タップ・下スワイプ終了
- 新規予定画面の入口統合
- 固定繰り返し＋任意間隔＋曜日指定
- 毎月の日付／第N曜日指定
- 複数お知らせ（か月・週・日・時間）
- 終日予定の通知時刻

## 操作LOCK
1. 日付1回目：選択。
2. 選択中の同じ日付を再タップ：新規追加。
3. 別日タップ：別日選択のみ。
4. 予定詳細：選択日の予定行から開く。
5. 予定バー：カレンダー上の表示専用。
6. シート上部を下へ72px超ドラッグ：閉じる。
7. シート外タップ：閉じる。

## 繰り返しLOCK
- 固定：なし、毎日、毎週、毎月、毎年。
- 毎週：複数曜日、平日、土日。
- 毎月：日付、第1〜第5曜日、最終曜日。
- カスタム：1〜99の間隔で日／週／か月／年。
- 終了：指定なし、終了日、1〜200回。

## お知らせLOCK
- 予定ごとに複数件。
- 数値＋か月前／週間前／日前／時間前。
- 個別削除可能。
- 終日通知時刻を設定可能。
- 設定値は各繰り返し予定へ引き継ぐ。

## 維持
- Plan-01.1の複数予定、開始・終了日時、種類・同行者カスタム。
- Record-02.2全機能。
- CLEAN v6デザイン。
