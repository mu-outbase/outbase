## 反映
- 起動直後は予定タブ、カレンダー主役。
- 予定はキャンプ・自宅散歩・ピクニック/軽量外出を同じ「予定→準備→記録→思い出」構造で扱う。
- 探すはキャンプ候補だけでなく、散歩場所、場内探索、帰路立ち寄り、ピクニック候補も候補として扱う。
- 準備は買い物 / 料理 / ギア / コタ / ルート / 天気 / 積み込み・出発前を予定ごとに分離。
- 当日運転席は下ナビに出さず、予定から開く文脈画面。
- ＋は写真/動画/声/メモ/GPS/あとで整理を、今の活動へ紐づける。
- 思い出は時系列 / 未確認箱 / 次回改善 / Google Photos / 復旧 / MVP監査カバーを持つ。
- AI/GPS推定は未確認候補に留め、勝手に確定しない。
- 予定・準備・記録・写真・メモ・GPS・コタ情報を勝手に修正/統合/上書き/削除しない。
