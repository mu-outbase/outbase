# OUTBASE BOOT v7.2.1 / Core07.2.1 準備司令塔UI微修正

## 目的
Core07.2 UI-FIXで近づいた大枠を維持しつつ、実利用で邪魔だった点だけを修正する。

## 修正内容
- キャンプ予定の並び順を日程が近い順へ変更
- 天気判断コピーを完全削除
- 料理 / ルート / ギア / コタ の各パネルに実用的な確認内容を追加
- 準備トップ通常時の下端余白を削減

## 維持する流れ
詳細天気を見る → 無料キャンセル期限までに判断 → 料理 / ルート / ギア / コタを必要なものだけ開く。

## 方針
天気は共有ではなく本人判断用。LINEコピーは買物・料理・出発だけに限定する。
