# OUTBASE BOOT v8.7 Core08-D5 当日ナビ型UI

## 目的
Core08-D4の工程一覧型を廃止し、当日タブを「今ここ」中心のナビ型UIへ変更する。
ユーザーがスクロールしながら工程を探し、使い方を覚え、判断して押す設計は禁止。

## LOCK方針
当日画面は、全工程を管理する画面ではなく、OUTBASEが予定・実績・GPS・時刻・準備内容から「今ここ」を推定して先回り表示する画面とする。

## 表示構成
1. 今ここカード
   - キャンプ名
   - 今ここ候補
   - 予定時間
   - 実績時間
   - ズレ
   - GPS状態
   - 次に見ること

2. すぐ残す
   - 写真
   - メモ
   - 音声
   - あとで
   - 動画と事前メモは小さく補助扱い

3. あとで整理する箱
   - 押し忘れ、曖昧メモ、写真、音声、動画を一旦集約
   - 料理・コタ・設営・撤収・買い出し・次回注意などの候補を表示
   - 自動確定は禁止

4. 今日の流れを見る / 今ここを直す
   - 普段は閉じる
   - 必要な時だけ工程一覧を開く

## データ保護
- dayRecords は追記型
- dayFlowState は実績時間のみ更新
- GPS・候補・推定は contextSnapshot / inferenceCandidates として保存
- ユーザー確認なしに分類・上書き・統合・削除しない
