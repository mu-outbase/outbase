# OUTBASE BOOT v8.4 Core08-D2 当日かんたん記録

## 目的
Core08-Dの当日画面は情報量が多く、現地で考えながら色々なボタンを押す必要が出てしまった。
Core08-D2では、当日画面を「工程管理」ではなく「押し忘れ前提の現地メモ箱」に戻す。

## LOCKルール
- 当日画面は正確に押す画面ではない。
- 押し忘れ・後追い・事前メモ・曖昧登録を前提にする。
- 最初から工程分類を選ばせない。
- 写真・メモ・音声・動画を大きいボタンで残す。
- 分類は任意タグとして後から付けられる扱いにする。
- AI整理は候補表示のみ。勝手に分類・上書き・削除しない。
- 当日記録 dayRecords は追記型。既存記録を壊さない。

## 画面構成
1. 今日のキャンプ名 / 記録件数 / 経過情報
2. 残し方切替：今すぐ / あとで登録 / 事前メモ / 何となく
3. 大きい4ボタン：写真 / メモ / 音声 / 動画
4. ゆるい追加：あとで登録 / 事前にメモ / 何となく残す
5. 今日の記録タイムライン
6. AI整理候補（表示のみ）

## 任意タグ
未分類 / 出発 / 到着 / 設営 / 料理 / 場内確認 / コタ / 撤収 / 買い物 / 気づき / 忘れ物 / 次回注意

## データ保護
- dayRecords は projectKey ごとに保存。
- 既存の dayRecords が patch から抜けた場合は復元保護する。
- 削除・統合・上書きはしない。
