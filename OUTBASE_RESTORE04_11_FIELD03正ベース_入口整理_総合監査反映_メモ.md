# OUTBASE RESTORE04.11 FIELD03正ベース 入口整理・総合監査反映

## 位置づけ
- 承認済み正ベース: RESTORE04.8 ルートAPIなし版
- RESTORE04.9/04.10は未承認のため積み増し正ベースにしない
- 04.11は04.8から、必要部品だけを再実装

## 目的
- 使いたい機能に迷わず入れる
- プランチップは機能入口ではなく、主役プラン確認/切替に限定
- キャンプと散歩の文脈を混ぜない
- 支払い確認サンプルを通常導線から除外
- 未確認箱は通常運用ではなく保険として扱う

## 実装概要
- 上バー: OBロゴ + 小型プランチップのみ。準備前/準備中/現地/帰宅後は構造側から出さない
- 下バー: 予定 / 探す / 準備 / 記録 / 思い出 に固定
- ホーム: 「何をする？」の大きな入口に再整理
- 記録: 話す / 撮る / 動画 / 場所 / ピン / メモに集約
- 散歩プラン: 散歩準備・散歩記録・GPS中心。キャンプ用ルート/チェックイン/SA/PA/設営撤収は非表示
- キャンププラン: RESTORE04.8のルートAPIなし版を維持
- 保存先: 主役プランに紐付け。不明時のみ未確認箱
