# OUTBASE BOOT v6.07 / Core06.07 使い勝手修正反映

## 位置づけ
Core06.07は、Core06.06で残っていた「操作できるが使いにくい」問題を潰すためのUX修正版。
カレンダーのタップ判定、日付位置、下ナビ視認性、記録モードの見つけやすさ、散歩マップ表示、記録中の強制遷移感を修正する。

## 反映内容
- カレンダーの日付セルは数字だけでなく枠全体をタップ対象にする。
- カレンダーの日付数字は左上固定にする。
- 下バーは濃色ベースに戻し、白文字で視認性を上げる。
- 記録中インジケーターはタップで強制的に記録画面へ戻らない非操作表示に変更。
- 記録画面は最上部に「今：モード名」を大きく表示。
- モード切替をスクロールなしのグリッド表示に変更。
- よく使うボタンをファーストビューに寄せる。
- 散歩マップはiframeを使わず、近距離ルートパネルとして常時表示する。
- 自宅散歩/キャンプ場散歩/キャンプ滞在/設営/料理/撤収/メモのモード表示を明確化。
- Service WorkerキャッシュをCore06.07に更新。

## 注意
- 記録中でも、準備・探す・当日・思い出へ移動してよい。自動で記録画面へ戻さない。
- 記録へ戻る場合は下バーの「＋記録」を押す。
