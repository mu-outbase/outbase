# OUTBASE RESTORE04.17 FIELD03正ベース トップカレンダー・散歩地図復帰

## 目的
04.16でルート画面は戻ったが、予定タブが外出タイプ選択画面になっており、OUTBASEのトップとして不適切だった。
04.17では、予定タブを「カレンダー / 今日 / 次の外出」中心に戻し、外出タイプ選択は新規予定の中へ下げる。
また、記録タブに散歩・現地用の地図導線を復帰させる。

## 修正内容
- VERSIONを `outbase-restore04-17-field03-calendar-map-flow-20260710` に更新
- 予定タブをカレンダー主役に変更
- 外出タイプ選択を常時表示から `＋新規予定の種類を選ぶ` の詳細内へ移動
- 今日の予定、次の外出、選択日の予定を予定タブに配置
- 記録タブに `散歩マップ` を主役表示として追加
- 散歩マップにGPS点数、距離、時間、平均速度、ピン数を表示
- 散歩マップから散歩開始、現在地、ピン、Google Mapへ進める導線を追加
- 準備タブでは「ルートは移動」「散歩地図は記録タブ」と役割分離
- 04.8系ルート出発逆算は維持
- Context Model / 外出属性モデルは内部維持

## 不採用
- 予定タブにキャンプ/通常散歩/イベントなどを大きく常時表示する構造
- Context Model説明カードの常時表示
- 散歩地図をルート画面に混ぜる構造
