# OUTBASE v144 全予定カレンダー＋文脈エンジン 一括実装LOCK

## 目的
v143仕様をもとに、OUTBASEを「全予定カレンダー＋どこから始めても予定/出来事へまとまる構造」にする。

## 重要
- 計画ファーストではない
- 記録ファーストではない
- キャンプ専用ではない
- 未整理に全部投げない
- Restart-42/43の別物UIは使わない
- 37〜40系の色・質感・小型チップ系の雰囲気に戻す

## 入れたもの
- 今日ホーム
- 全予定カレンダー
- 5入口追加：予定 / 準備 / 現地 / 実績 / メモ
- 予定/出来事詳細
- 準備・記録・改善
- 文脈自動紐付け 初期版
- 要確認 少数確認
- バックアップコピー/読み込み

## まだ本接続しないもの
- GoogleカレンダーAPI
- ジョルテ独自データ直接読込
- Google Photos
- 天気API
- AI完全判定
- 家族共有

https://mu-outbase.github.io/outbase/?v=v144-calendar-context-lock

## 確認ポイント
1. ホームが「今日・次・要確認・すぐ追加」になっている
2. カレンダーで全部の予定を見られる
3. 予定を追加できる
4. 準備から始めても予定/出来事に入る
5. 現地記録から始めても予定/出来事に入る
6. 過去実績を作れる
7. 要確認が全件整理箱になっていない
