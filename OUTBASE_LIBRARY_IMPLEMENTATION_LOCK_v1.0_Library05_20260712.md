# OUTBASE Library 実装LOCK v1.0

## 目的
OUTBASEの資産機能を、単純な持ち物一覧ではなく、準備・使用・在庫・整備・履歴へつながるキャンプ資産管理として実装する。

## 画面構成
### 資産トップ
- 総資産
- 補充必要
- 次回整備
- 今回の予定
- 今やること
- お気に入り
- 最近の履歴

### ギア内部
- アイテム
- セット
- カスタム
- 在庫
- つながり
- 履歴

### その他
- 車・自転車
- ペット
- 保管場所
- 取込・出力

## データ原則
- 単品とセットを混同しない
- 保管場所と一緒に使う関係を混同しない
- カスタムパーツは単品として所有し、構成として装着状態を保存する
- 消耗品・燃料は数量、単位、補充ライン、使用先を持つ
- 履歴は対象種別をまたいで時系列管理する
- 数百〜数千件を前提に検索・分類・段階表示を使う

## UI原則
- 深いグリーンと真鍮色を基調とする
- 管理表ではなく資産庫として見えること
- 最初に全件ではなく、今見るべき情報を出す
- スマホ片手操作を優先する
