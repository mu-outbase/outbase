OUTBASE v166.31 R1-02
HOME統一・予定詳細即時復帰 v18.1

基準
- GitHub main v18.0 を土台にした差し戻し修正版
- GitHub main はこのZIP作成時点では変更していない
- Android実機の最終確認前。R1完了宣言はしない

今回まとめて修正した内容
1. 準備画面をHOME v36 r34の視覚文法へ統一
   - HOMEと同じ背景色、白カード、境界線、影、角丸、文字密度、アイコン表現を使用
   - 大型の緑色ヒーローカードを廃止
   - カード内カードを減らし、準備項目を一つの一覧として整理
   - 不要な画面内コピーライトと追加スペーサーを撤去

2. 予定詳細への復帰を即時化
   - 予定詳細の表示モデルを90秒間メモリ保持
   - 準備へ移動する前の予定詳細を保持
   - 準備表示中に予定詳細データをバックグラウンドで先読み
   - 戻る操作では遷移アニメーションを待たず、保持済み画面を先に描画

3. 主役プラン表示を整理
   - 現在の主役プランは全画面で確認可能
   - 切替候補が複数ある時だけ「切替」を表示
   - 候補が一つだけの場合は、切替ボタンではなく現在予定の表示として扱う
   - 完了・アーカイブ済み予定は通常の切替候補から除外
   - 候補の読込は起動描画を妨げないアイドル時間に実行

維持したもの
- FIELD03本体
- GPS、地図、写真、動画、音声、メモ、ピン
- 画面ON維持、IndexedDB、オフライン復元
- HOME v36 r34の天気

- 予定詳細v16.3の画像・間隔

実機で見るポイント
- 準備画面がHOMEと同じ白カード基調に見えること
- 予定詳細へ戻る操作が体感で即時であること
- 主役予定が一つだけなら「切替」が出ないこと
- 活動を始める／詳細な準備を開くが従来どおりFIELD03へ接続すること
