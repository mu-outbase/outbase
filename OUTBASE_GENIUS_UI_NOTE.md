# OUTBASE GENIUS UI

## 方針
「カッコいい」と「便利」を同時に狙った再設計。
新しい技術: CSS Grid / sticky glass header / localStorage PWA / Canvas route map / touch swipe calendar
古い技術: 黒・真鍮・木・紙の道具感、コントロールパネル的な構成

## 主な変更
- ホームを宣伝コピーではなく、Control Boardへ変更。
- 画面上部にステージ切替（準備前 / 準備中 / 現地 / 帰宅後）。
- カレンダーは単体・連日・キャンプ・繰り返しを視覚的に分離。
- 準備は料理 / 買い物 / ギア / 天気。
- ギアは一覧 / ボックス / 積込 / 乾燥修理 / 取込。
- 現地は散歩GPS / 簡易地図 / スポット / 犬友達 / 非公開体調メモ。
- うんち/おしっこは表に出さず、非公開体調メモ内のみ。


## scroll fix
- `.shell { overflow:hidden }` がモバイルChromeで sticky header を壊していたため解除。
- sticky header 側に角丸と z-index を持たせ、スクロール時もヘッダーが消えないように修正。
- body背景の途中から黒帯が見える問題を避けるため、外側背景を安定した黒に変更。
- モバイルではヘッダー角丸を消して、スクロール時に自然に固定されるよう調整。


## scroll clean fix
- Android Chromeでスクロール時に上部ヘッダーがコンテンツへ被り、画面が途中で切れて見える問題を避けるため、上部ヘッダーを sticky から通常フローへ変更。
- 黒いサイド余白・途中の黒帯が目立たないよう、外側背景を安定した生成りへ変更。
- `.shell` の影と角丸を抑え、スクロール中にアプリ本体が額縁の中でズレて見える問題を修正。


## app-frame scroll fix
- ページ全体スクロールをやめ、アプリ内部の `.main` だけがスクロールする構造へ変更。
- 上部ヘッダーと下部ナビは常に見える。
- スクロール中に上部ヘッダーが半分切れて見える問題を修正。
- Chromeアドレスバーの開閉で画面が変にズレる問題を抑制。


## EASYFLOW
- 見た目OK版を崩さず、押す場所を減らした。
- ホームを「次に押す場所」中心へ変更。
- 準備チェックをホーム上で確認できるようにした。
- 現地画面は散歩開始・現在地・スポット・犬友達・メモを大きなボタンに整理。
- 体調メモは引き続き非公開パネル内のみ。
- 詳細設定は必要な時だけ開く。


## LINKED CORE
- 見た目OK/EASYFLOWを維持し、予定→準備→現地→整理のつながりを強化。
- ホームと準備に「つながり」パネルを追加。
- 準備まとめコピーを追加（買い物・未積込ギア・天気判断を1つにする）。
- 「次に進める」ボタンを追加し、買い物→ギア→天気→現地へ自動誘導。
- 全ギア積込済みの一括操作を追加。
- 思い出に自動レビュー生成・保存・コピーを追加。


## CALENDAR PRO
- カレンダーを「見るだけ」から「その場で動かす」画面へ強化。
- 今日へ戻る、予定追加、次へ進むを月表示直下へ追加。
- 選択日の下に、キャンプ / 予定 / 散歩 / 支払いのワンタップ追加を追加。
- 日付未設定の予定を、選択中の日付へすぐ入れられるように変更。
- 今月の流れをタイムラインで表示。
- 予定詳細から「主役にする」を追加。
- キャンプ日には、天気未確認・ギア未積込の注意を表示。
