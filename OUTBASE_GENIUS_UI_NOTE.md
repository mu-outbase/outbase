# OUTBASE GENIUS UI

## 方針
「カッコいい」と「便利」を同時に狙った再設計。
新しい技術: CSS Grid / sticky glass header / localStorage PWA / Canvas route map / touch swipe calendar
古い技術: 黒・真鍮・木・紙の道具感、コントロールパネル的な構成

## 主な変更
- ホームを宣伝コピーではなく、Control Boardへ変更。
- 画面上部にステージ切替（準備前 / 準備中 / 現地 / 帰宅後）。
- カレンダーは単体・連日・キャンプ・繰り返しを視覚的に分離。
- 準備は料理 / 買い物 / ギア / 天気。
- ギアは一覧 / ボックス / 積込 / 乾燥修理 / 取込。
- 現地は散歩GPS / 簡易地図 / スポット / 犬友達 / 非公開体調メモ。
- うんち/おしっこは表に出さず、非公開体調メモ内のみ。


## scroll fix
- `.shell { overflow:hidden }` がモバイルChromeで sticky header を壊していたため解除。
- sticky header 側に角丸と z-index を持たせ、スクロール時もヘッダーが消えないように修正。
- body背景の途中から黒帯が見える問題を避けるため、外側背景を安定した黒に変更。
- モバイルではヘッダー角丸を消して、スクロール時に自然に固定されるよう調整。


## scroll clean fix
- Android Chromeでスクロール時に上部ヘッダーがコンテンツへ被り、画面が途中で切れて見える問題を避けるため、上部ヘッダーを sticky から通常フローへ変更。
- 黒いサイド余白・途中の黒帯が目立たないよう、外側背景を安定した生成りへ変更。
- `.shell` の影と角丸を抑え、スクロール中にアプリ本体が額縁の中でズレて見える問題を修正。


## app-frame scroll fix
- ページ全体スクロールをやめ、アプリ内部の `.main` だけがスクロールする構造へ変更。
- 上部ヘッダーと下部ナビは常に見える。
- スクロール中に上部ヘッダーが半分切れて見える問題を修正。
- Chromeアドレスバーの開閉で画面が変にズレる問題を抑制。


## EASYFLOW
- 見た目OK版を崩さず、押す場所を減らした。
- ホームを「次に押す場所」中心へ変更。
- 準備チェックをホーム上で確認できるようにした。
- 現地画面は散歩開始・現在地・スポット・犬友達・メモを大きなボタンに整理。
- 体調メモは引き続き非公開パネル内のみ。
- 詳細設定は必要な時だけ開く。


## LINKED CORE
- 見た目OK/EASYFLOWを維持し、予定→準備→現地→整理のつながりを強化。
- ホームと準備に「つながり」パネルを追加。
- 準備まとめコピーを追加（買い物・未積込ギア・天気判断を1つにする）。
- 「次に進める」ボタンを追加し、買い物→ギア→天気→現地へ自動誘導。
- 全ギア積込済みの一括操作を追加。
- 思い出に自動レビュー生成・保存・コピーを追加。


## CALENDAR PRO
- カレンダーを「見るだけ」から「その場で動かす」画面へ強化。
- 今日へ戻る、予定追加、次へ進むを月表示直下へ追加。
- 選択日の下に、キャンプ / 予定 / 散歩 / 支払いのワンタップ追加を追加。
- 日付未設定の予定を、選択中の日付へすぐ入れられるように変更。
- 今月の流れをタイムラインで表示。
- 予定詳細から「主役にする」を追加。
- キャンプ日には、天気未確認・ギア未積込の注意を表示。


## PREP PRO
- 準備を「管理」ではなく「終わらせる」画面へ再構成。
- 次にやることを自動表示。
- 準備スコア、料理数、買い物残、積込数、天気確認を上部で一目表示。
- 料理から材料を買い物へ追加。
- 料理から必要ギアの不足候補を追加。
- 買い物をグループ表示に変更。
- 購入済み整理を追加。
- 天気判断を、設営 / 撤収 / コタ / 幕体のメモとして保存。
- ギア積込画面に積込順とリセットを追加。


## GEAR PRO
- ギア管理を「台帳」ではなく「積める・戻せる・直せる」画面へ強化。
- ギアスコア、積込数、ケア対象、BOX未設定数、BOX数を上部表示。
- ギア検索を追加（ギア名 / BOX / 家の場所 / 車載位置 / 状態）。
- BOX画面を、家の保管場所・車載位置・中身・積込数が見える構成へ変更。
- BOX単位で積込済み / 家へ戻す / 乾燥へ / 中身コピー。
- 積込画面をBOX単位の操作へ強化。
- 乾燥/修理/買替キューを追加し、完了/修理/買替の状態変更を追加。
- ギアまとめコピーを追加。


## FIELD PRO
- 現地記録を「考えず3秒」で残す画面へ強化。
- 散歩開始 / 現在地 / メモ / 写真 / 音声 / 料理を大きなボタンに整理。
- 通常散歩 / キャンプ場散歩を維持。
- 景色 / 木陰 / 水場 / 危険をワンタップのスポット記録へ追加。
- スポットは場所カードにも自動追加。
- 現地タイムラインを追加。
- 現地まとめコピーを追加。
- 犬友達を公開記録として残す。
- 体調 / うんち / おしっこ は非公開パネル内だけ。現地まとめ・場所カード・レビューへ混ぜない。


## MEMORY PRO
- 思い出/整理を「残すだけ」から「次回使える形」へ強化。
- レビュー / 記録 / 場所 / 改善 / メモ / データのタブを追加。
- 公開記録だけでレポート生成。体調 / うんち / おしっこは除外。
- レポート保存・コピー。
- 記録から改善メモへ送る。
- 記録から場所カードへ送る。
- 場所カードのコピー / 再訪+1 / 予定化。
- 改善メモを次回準備の買い物へ送る。
- バックアップ / 復元 / 整理をデータ画面へ集約。


## DISCOVER PRO
- 抜けていた「探す」を下部ナビへ追加。
- 候補探しを、条件確認・保存・比較・予定化まで一気通貫に強化。
- キャンプ候補 / 散歩候補 / 保存候補 / 条件 / 比較タブを追加。
- 犬可 / 温水 / ドッグフリー / 景色 / 直営 / 乾燥 / 距離などを点数化。
- 候補から予定へワンタップ追加。
- 候補追加・保存切替・候補コピー・上位比較コピーを追加。
- ホームは上部OUTBASEロゴから戻れるよう維持。


## CENTER PRO
- ホームを「全体センター」として強化。
- 予定 / ギア / BOX / 買い物 / 料理 / 場所 / 候補 / メモ / 記録 / 天気を横断検索。
- 未整理・未完了ボックスを追加。
- 日付未設定予定、買い物未完、ギア未積込、天気未確認、保存候補をまとめて処理。
- 現在地コピーを追加。
- 迷ったら「次に進める」で、買い物→ギア→天気→現地へ誘導。


## AUDIT PRO
- 仕上げ前の総合監査と安定化。
- ホームとデータ画面に総合監査パネルを追加。
- 主役プラン、日付未設定、買い物未完、ギア未積込、BOX未設定、天気未確認、ケア対象、非公開混入を監査。
- 非公開体調メモが公開記録に混じった場合、自動補修で private 化。
- BOX未設定ギアの自動補修。
- 検索入力中に画面全体が再描画されて入力が途切れる問題を修正。
- サービスワーカー登録を追加し、PWA/オフライン動作を安定化。
- 監査コピーを追加。


## MEDIA PRO
- 写真・動画・音声を「撮った前提」ではなく、実際の記録として扱えるように強化。
- 現地画面に写真ファイル選択、動画ファイル選択、音声録音を追加。
- 小さめの写真/動画/音声はプレビュー付きで保存。
- 大きい動画はファイル名・サイズを記録し、localStorage肥大を避ける。
- 現地タイムラインと整理/記録タブでメディアプレビュー表示。
- 音声録音非対応やマイク拒否時は文字メモ保存へフォールバック。
- 通常の復元/取込とメディア取込が衝突しないよう importMode を追加。


## WEATHER PRO
- 天気を「見る」ではなく、設営 / 撤収 / コタ / 幕体の判断へ変換。
- 14日前 / 7日前 / 3日前 / 前日 / 当日の確認タイミングへ拡張。
- 予報メモを、設営 / 夜 / 朝 / 撤収の場面で管理。
- 降水確率 / 風速 / 気温 / 湿度からリスク点を表示。
- 設営判断、撤収判断、コタ判断、幕体判断を個別に保存。
- 天気ソースと更新時刻を記録。
- 天気まとめコピーを追加。
- 準備まとめにも天気判断レポートを統合。
- 総合監査に天気判断未決定を追加。


## FAMILY PRO
- 家族・ペットを、準備 / 現地 / 共有へ接続。
- ペット情報（コタ / アオ / エラ / ユキ）を追加。
- ペット準備チェックを追加。
- コタ用品を買い物へ反映。
- ペット準備を準備画面のタブに追加。
- 思い出/整理にも家族タブを追加。
- リン共有用コピーを追加。
- 共有履歴保存を追加。
- 共有文から体調 / うんち / おしっこ等の非公開メモを除外。
- 総合監査にペット準備未完を追加。
- 全体検索にペット/ペット準備を追加。


## CONNECT PRO
- 静的PWAで現実に動く外部連携を強化。
- 整理画面に「連携」タブを追加。
- ICSカレンダー書き出し。
- Google Calendar追加画面を開く。
- ギアCSV / 買い物CSV / 場所CSV 書き出し。
- 端末共有API / クリップボード共有。
- リン共有 / 現在地コピー / レポートコピーを連携画面へ集約。
- JSONバックアップ/復元を連携画面にも整理。
- Google Photos / Google Calendar双方向同期 / 家族リアルタイム同期は静的PWA単体では未接続であることを明示。
- 総合監査に連携未確認を追加。


## START PRO
- 初回設定・テンプレート・本番開始を強化。
- ホームに「はじめる」パネルを追加。
- キャンプテンプレート作成。
- 散歩テンプレート作成。
- 支払いテンプレート作成。
- 本番データへ切替。
- スターターバックアップ。
- データ移行/補修を load 時にも実行。
- 古いバージョンの保存データでも、pets / weatherPlan / records / gear / boxes などを補完。
- データ画面にSTARTPRO_MIGRATIONを追加。
- 総合監査にスターター未完を追加。


## MEAL PRO
- 料理を「献立メモ」から「買い物・ギア・調理順」へ接続。
- 料理タブ内に、献立 / 調理順 / テンプレ / 注意 / 共有 を追加。
- ガーリックシュリンプ、手作りピザ、ホットドッグ、たたききゅうり、豚しゃぶテンプレートを追加。
- 貝アレルギー / 魚卵苦手チェック。
- 献立をLINE用にコピー。
- 全材料を買い物へ一括反映。
- 現地調理順を生成。
- 料理を現地タイムラインへ記録。
- 買い物コピーをグループ表示化。
- 総合監査に料理注意を追加。


## PLAN PRO
- 主役プラン切替を全画面上部から使いやすく強化。
- 上部のプラン名タップで、邪魔にならない切替ドロワーを開く。
- プランごとの準備 / ペット / 記録 / 整理のミニ進捗を表示。
- ホームに主役プランボードを追加。
- 主役切替時に選択日・表示月も同期。
- 現地記録、現地まとめ、整理画面の公開記録、レビュー生成を主役プラン基準へ変更。
- 主役プランのコピーを追加。
- 総合監査に主役記録なしを追加。


## TIMER PRO
- タイマー機能を追加。
- 設営 / 料理 / コタ / 撤収 / 乾燥 / 散歩 / 自由タイマー。
- 主役プランごとのタイマー表示。
- ホーム / 準備 / 現地にタイマーを表示。
- 開始 / 一時停止 / リセット / 完了。
- 完了時に現地タイムラインへ自動記録。
- タイマーまとめコピー。
- 全体検索にタイマーを追加。
- 総合監査にタイマー実行中を追加。
