# OUTBASE RESTORE03.6 FIELD03正ベース APIなしMVP整理

## 正ベース
- OUTBASE_RESTORE03_5_FIELD03正ベース_予定連動出発逆算.zip
- FIELD03土台維持。REBUILD17/18/18.1/RESTORE01は使っていない。

## 方針
APIなし版は「ルート検索アプリ」ではなく「キャンプ当日の自宅出発時間を決める道具」として整理。

## 修正
- 画面を4ステップに整理：予定 / 移動 / 寄り道 / 出発。
- 保存済み出発計画を上部に強く表示。
- 保存済み計画から「この計画で出発」を押しやすくした。
- 入力欄をまとめ、予定・自宅・当日想定移動時間・到着目標・寄り道だけに整理。
- 計画内訳は折りたたみにして、通常画面を軽くした。
- 自宅出発時間を結果カードとして強調。
