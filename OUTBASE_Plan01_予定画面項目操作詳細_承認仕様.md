# OUTBASE Plan-01 予定画面 項目・操作・詳細 承認案 v1

作成日：2026-07-11
対象：CLEAN v6 / Record-02.2 継承

## 1. 正本上の役割

- 予定タブは、外出だけでなく日常予定も含む「全予定カレンダー」。
- 予定が先にあるものだけでなく、今始まった出来事、過去の写真・メモから追加する実績も同じ画面で扱う。
- 予定・出来事詳細を、準備・当日記録・思い出整理へつながる親画面とする。
- カレンダーは主入口だが、唯一の入口にはしない。

## 2. 予定メイン画面

### 月カレンダー
- 月移動：前月 / 今日 / 次月
- 「＋予定」：予定・出来事追加シートを開く
- 日付1回タップ：選択日を変更
- 空いている日を2回タップ：その日付で追加シートを開く
- 予定バータップ：予定・出来事詳細を開く
- 複数日予定を週単位で分割表示
- 普通の予定、散歩、ドライブ散歩、キャンプ、登山、イベント、買い物、日常を同じカレンダーへ表示

### 選択日の予定
- 選択した日だけを表示
- 時間 / 種類 / 同行者 / 予定名 / 準備状態を表示
- 「＋追加」で選択日へ追加
- 「すべての予定を見る」で全予定一覧を開く

## 3. 予定・出来事の追加

追加入口は3つに分ける。

1. 予定
   - これからの予定として作成
2. 今から
   - 予定がなかった出来事を開始時点で作成
3. 過去から
   - 過去の写真・メモ・実績から出来事として作成

入力項目：
- 名前
- 開始日
- 終了日
- 時間
- 種類
- 同行者
- 場所
- メモ

最初から全項目を必須にしない。名前と日付があれば追加でき、詳細は後から補完する。

## 4. 予定・出来事詳細

表示：
- 予定名
- 種類
- 日付・時間
- 状態
- 場所
- 同行者
- 準備状態
- メモ

操作：
- この予定を主役にする
- 準備を始める
- この予定で記録
- 編集
- 完了にする / 予定に戻す
- 削除

「準備を始める」は対象予定を主役に設定して準備へ移動する。
「この予定で記録」は記録先をその予定へ設定して記録画面へ移動する。
単なるタブ移動ではなく、対象予定の設定を同時に完了する。

## 5. すべての予定・出来事

- 日付順で一覧表示
- 日付範囲 / 時間 / 種類 / 状態 / 予定名を表示
- 行タップで詳細
- 一覧から追加可能

## 6. データ方針

Plan-01 MVPでは次を保存する。
- id
- title
- type
- start / end
- time
- companion
- location
- status
- prep
- note
- tone

ブラウザ再読込後も保持する。
今後、project / experience / session / record へ正式接続する際も、予定・出来事の見た目と操作は維持する。

## 7. 維持・禁止

維持：
- CLEAN v6のクリーム / 深緑 / 真鍮ゴールド
- 小型プランチップ
- Record-02.2の実GPS、音声、写真、動画、メモ、ピン、終了復旧、破棄保護
- FIELD03正ベース

禁止：
- カレンダーを外出予定だけに限定しない
- 選択日の予定に別日を混ぜない
- 巨大な＋だけを主役にしない
- 説明カードを大量に並べない
- 予定作成時に全項目入力を要求しない
- 主役操作を単なる別タブ遷移にしない
