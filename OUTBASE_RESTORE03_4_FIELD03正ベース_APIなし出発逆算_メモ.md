# OUTBASE RESTORE03.4 FIELD03正ベース APIなし出発逆算

## 正ベース
- OUTBASE_RESTORE03_3_FIELD03正ベース_ルート先行記憶.zip
- FIELD03土台維持。REBUILD17/18/18.1/RESTORE01は使っていない。

## 方針変更
APIなし版では、寄り道を細かく検索・分析しない。
寄り道の有無と所要時間だけ管理して、チェックイン時間から自宅出発時間を出す。

## 修正
- ルート画面を「APIなし出発逆算」に整理。
- Google Mapsで移動時間を確認し、その分数をOUTBASEへ入力。
- チェックイン、到着余裕、遅刻許容、渋滞/トイレ余裕を入力。
- 寄り道は行く/行かない＋所要時間だけ。
- 寄り道項目：給油、コンビニ、スーパー、SA/PA休憩、コタ休憩、道の駅、観光。
- 自宅出発時間を自動計算。
- ルート計画として保存。
- 当日は保存済みルートからGoogle Maps確認＋OUTBASE記録へ。

## APIなしではやらない
- 高速名の自動取得
- 入口IC/出口IC/JCTの自動取得
- SA/PAサービスの自動取得
- 実店舗候補の自動提案
- ルート上判定の自動計算

## 触らない範囲
- 現地セッション
- 地図/GPS
- 写真/動画/音声
- ピン
- 欠測区間
- 画面ON維持
- バックアップ/診断
- 非公開体調メモ
- ギア/料理/天気の既存実作業画面
