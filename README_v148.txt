# OUTBASE v148 前進修正 37-40デザイン整合LOCK

## 目的
v147で入れた全予定MVP機能は捨てず、見た目を37〜40系へ戻す前進修正。

## 修正
- 戻しZIPではない
- v147機能は残す
- トップを 37〜40 の「OUTBASE / RESTART-35」「今日は何する？」寄りに戻す
- 下ナビの絵文字を廃止
- ＋を巨大・二重表示にしない
- 日付なし予定をカレンダー日付に出さない
- カレンダー点を最大3個に制限
- 日付未設定枠を分離
- v147のデザイン逸脱を抑える
