# OUTBASE MVPβ Core04 改善タブ実用化

Core03.1で通過した記録・履歴・詳細画面の次段階として、改善タブを実用化する。

## 目的
記録で出た「暑さ」「タオル不足」「雨撤収」「コタ用品」などを、次回の買い物・持ち物・コタ用品・反省へ戻す。

## 主な変更
- 改善タブを表示だけではなく、準備へ反映できる画面に変更
- 改善キューを分類表示
- 個別反映 / まとめて反映
- 次のキャンプカードの prep に統合
- 反映済みログを保持
- ホームに改善キュー件数を表示
- service-worker キャッシュ名をCore04へ更新

## 判定
改善タブで記録由来の改善候補を準備へ反映し、準備タブで買い物・持ち物・コタ用品・反省に追加されることを確認する。
