# OUTBASE MVPβ Core05.9 / Jorte-style Calendar UX

カレンダー機能をジョルテの操作感に寄せた版。

## 主な変更
- 月間カレンダーを主画面化
- 日付セル内に予定名を直接表示
- 土日・祝日の色分け
- 予定種別ごとの色分け
- 日付タップで下部の ToDo&メモ 詳細欄を表示
- 下部詳細欄から普通予定・キャンプ・誕生日・車検・仕事・ToDoを登録
- ToDo完了チェックとグレー表示
- 誕生日など毎年繰り返し
- 当日/前日/1週間前通知
- 左右スワイプと前月/翌月ボタン
- 今日ボタン、新規ボタン、通知ONボタン
- ＋記録を下ナビ内へ統合

## 注意
ブラウザ通知は、アプリ起動中または起動時チェックを基本とする。アプリを完全に閉じた状態での確実な通知は、後続のPWA通知強化フェーズで扱う。
