# OUTBASE MVPβ Core05.11 / Jorte Refined Confirm

Core05.11 は、ジョルテ型カレンダー機能を維持しつつ、Core05.7寄りの生成り・紙感デザインを継続した版。

## 主な反映
- 下部バーの「今日」と「予定」重複を整理し、下部バーを「予定 / 探す / 準備 / ＋記録 / 当日 / 思い出」に統一。
- カレンダー内の「今日」ボタンで今日へ戻る。
- 日付1回タップで ToDo&メモ、同じ日付をもう1回タップで予定追加を開く。
- 予定登録で開始時間・終了時間を任意登録可能。
- 繰り返しは、なし / 毎週 / 毎月 / 毎年 / 数年おきに対応。
- 連続予定はカレンダー上で繋がるバーとして表示。
- ToDo&メモの予定ラベルをタップすると登録内容の詳細を表示。
- 予定削除は確認ダイアログを挟んで誤削除を防止。
