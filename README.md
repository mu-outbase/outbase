# OUTBASE MVPβ Core03.1 記録UX修正

Core03実機確認で判明した「詳細を見るボタンが動かないように見える」問題を修正する。

## 修正内容
- 詳細を見る押下後、詳細カードへ自動スクロール
- 選択中の履歴カードを薄緑でハイライト
- 詳細カードに「選択中の履歴カード」表示を追加
- タイトル入力欄をスマホ幅で広くするため field クラスを適用
- メモ欄に field/textarea クラスを適用して詰まりを改善
- ホーム文言をCore03.1へ更新
- service-workerキャッシュ名をCore03.1へ更新

## クリア判定
Core03.1反映後、記録終了→履歴カード→詳細を見るで、詳細画面へ移動することを確認する。
