# OUTBASE MVPβ Core05.5 Time & Phase UX

## 目的
Core05.4の実機確認で残った「アプリっぽくない」「色使いが弱い」「ホーム/準備/記録/改善の仕分けが分かりにくい」という課題に対応する。

## 変更方針
- 時間帯テーマ：morning / day / sunset / night を端末時刻で自動適用
- フェーズ整理：次 / 前 / 今 / 後 を主導線として表示
- 下ナビ整理：ホーム/準備/記録/改善の裏に、次/前/今/後の意味を明示
- ホーム：次のキャンプ・現在フェーズ・状態3つだけに絞る
- 準備：キャンプ前として、買う/持つ/食べる/コタに整理
- 記録：現地で残す画面として継続
- 改善：キャンプ後に次回へ戻す画面として継続

## 判定
まだ完成宣言ではない。Core05.5はデザイン方向性と導線認識の確認版。
