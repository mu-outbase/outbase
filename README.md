# OUTBASE MVPβ Core05.8 Interactive Calendar UX

目的：OUTBASEのホームを「普段のカレンダーアプリの代わり」に近づける。

## 反映内容

- 予定ホームのカレンダー操作を強化
- 左右スワイプで前後月へ移動
- 前月・翌月ボタン追加
- 日付タップで選択日パネルを表示
- 選択日から予定を直接追加
- 予定の開始日・終了日を設定可能
- 2日以上の予定は複数日に色付け
- 予定種類別カラー
  - キャンプ
  - 普通の予定
  - 誕生日
  - 車検・車
  - 仕事
  - その他
- 土日色分け
- 祝日表示（ローカル祝日テーブル）
- 通知ONボタン追加
- 誕生日・車検などのリマインダー項目を登録可能
- 下ナビ中央に＋記録を統合し、画面への被りを解消

## 注意

MVP段階の通知は、ブラウザ通知許可＋アプリ起動時/起動中確認を前提とする。
バックグラウンドで確実に鳴る通知は、今後のPWA通知強化フェーズで扱う。
