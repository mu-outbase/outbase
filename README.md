# OUTBASE MVPβ Core05.6 Calendar Home + 6 Navigation

## 目的
OUTBASEのホームをカレンダー中心へ戻し、4タブでは曖昧だった機能を6分類に再整理する。

## 6分類
- 予定：カレンダー、次のキャンプ、今日やること
- 探す：キャンプ場探し条件、犬可、温水、距離、料金
- 準備：買い物、持ち物、料理、コタ用品、天気、ギア
- 当日：出発、ルート、設営、夜ごはん、撤収
- 記録：写真、動画、音声、メモ、GPS、コタ散歩
- 思い出：記録、振り返り、改善、次回準備へ戻す

## UI方針
- ホームは月間カレンダー + 次のキャンプ + 今日やること
- 下ナビは5分類、記録は中央フローティングボタン
- 時間帯テーマは背景とアクセントに限定
- 機能色は固定し、現在地を見失わない
- 1画面1目的、主ボタン1つ、補助操作は折りたたむ
