# OUTBASE MVPβ Core03 記録入口

Core02.2で準備候補を通過したあと、キャンプ当日・コタ散歩・日常メモを記録する入口を追加する。

## 目的

- キャンプ前だけでなく、当日・散歩・日常でもOUTBASEを開く理由を作る
- 写真・動画・音声メモ・テキストメモ・GPS・タイマーを1つのセッションに束ねる
- 終了後に履歴カードと詳細画面を表示する
- 記録から次回改善キューを作る

## Core03の成功判定

- 記録タイプを選んで開始できる
- タイマーが動く
- GPS地点を記録できる
- メモ/写真/動画/音声メモを記録カード化できる
- 終了して履歴カードができる
- 詳細画面で記録一覧を見られる
- 改善キューに反映される

## 注意

- PWA本格確認はまだ行わない
- 写真/動画の本保存・クラウド連携は次ゲート
- Core03は記録入口の実用確認まで
