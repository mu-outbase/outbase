# OUTBASE MVPβ Core05.10 / Jorte Functions + Refined Design

Core05.10 は、Core05.9 のジョルテ型カレンダー機能を残しつつ、Core05.7 の生成り・紙・セージ・ブラス系の見た目へ戻した版。

## 反映内容
- ジョルテ型の月間カレンダー機能は継続
- 見た目は Core05.7 寄りの柔らかい OUTBASE デザインへ調整
- 予定登録に任意の開始時間・終了時間を追加
- 繰り返し予定を追加
  - なし
  - 毎週
  - 毎月
  - 毎年
  - 数年おき
- ToDo&メモのラベルをタップすると選択日の詳細へ移動
- カレンダーの日付タップ挙動を変更
  - 1回目：ToDo&メモ表示
  - 同じ日付をもう1回：予定追加フォームを開く
- 連続した予定は、カレンダー上でつながるバー表示に変更
- 複数日キャンプ予定も連続表示
- 土日・祝日・予定種別色分けは継続
- 通知はブラウザ通知＋アプリ起動時チェック
