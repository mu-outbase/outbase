# OUTBASE FIELD03 5画面一画面統合 v1

## 今回の変更
予定・探す・準備・記録・思い出の5画面を、最初の表示で判断と主要操作が完結する設計へまとめます。

### 直接Activity切替
- 各画面上部に「現在」のActivityボタンを横並び表示
- Activityボタンを押すと、その場で直接切替
- 現在使用中は濃緑
- 休止中は白＋破線
- ＋だけが新しい活動開始
- 旧「活動を切り替える」カードと6タブ画面は通常操作から除外

### ボタンとラベルの区別
- 押せるものは塗り・枠・動詞のいずれかを必須化
- 状態表示は背景なし・操作不可
- 旧フローティング虫眼鏡と下部Activityバーは非表示

### 5画面の一画面化
- 予定：今すぐカードを除外し、カレンダーと選択日の予定を主役化
- 探す：巨大な装飾ヒーローを除外し、検索入口を主役化
- 準備：重複する概要・コマンドバーを整理し、5項目を2列表示
- 記録：巨大なCOMMON LIFECYCLEカードを除外し、地図と操作を主役化
- 思い出：レビューを詳細側へ移し、記録・改善・場所・関連付けを主役化

画面内の「すべて見る」で詳細表示へ切り替えられます。
