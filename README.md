# OUTBASE MVPβ Core05.7 Calendar Home Refined

## 目的
Core05.6で採用したカレンダーホームを維持しつつ、実機確認で出た以下の違和感を修正する。

- 色使いが良くない
- 夜テーマが重く、アプリらしさより試作感が出る
- 次キャンプカードが大きすぎる
- 「今日」の下に下部ナビと同じ項目が並び、重複している

## 方針
- カレンダーはホームの主役として継続
- 時間帯テーマは全体を大きく変えず、アクセントだけに限定
- 次キャンプは大きなヒーローではなく、コンパクトな予定カードへ変更
- ホームの下部重複メニューを廃止
- 今日やることは「料理を決める」「買い物確認」「コタ用品確認」など実行項目にする
- 下ナビは 予定 / 探す / 準備 / 当日 / 思い出、記録は中央FABを継続
