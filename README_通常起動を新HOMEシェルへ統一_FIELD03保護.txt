OUTBASE 通常起動入口修正

根本原因
- 旧実装では ?shell=1 がある時だけ新シェルを起動
- 通常URL、ホーム画面追加、PWA起動では旧アプリを起動
- そのため共通シェル統合後も旧カレンダーへ戻っていた

修正
- 通常起動は自動的に ?shell=1&view=home へ正規化
- HOMEからカレンダーを開くと本体共通シェル統合版を使用
- 明示的な旧URL（tab/hash）は維持
- recordState=active/paused の現地記録中はFIELD03を維持
- IndexedDB、GPS、地図、写真、音声、天気r34は変更なし

確認手順
1. OUTBASEを完全終了
2. 通常のホームアイコン／ブックマークから起動
3. HOME表示
4. HOMEのカレンダーを開く
5. 上下バーがHOMEのまま中央だけカレンダーへ切り替わることを確認
