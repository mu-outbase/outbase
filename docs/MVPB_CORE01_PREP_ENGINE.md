# OUTBASE MVPβ Core 01 — 次のキャンプ準備エンジン

## 目的

予約スクショ・予約メール・PDF本文・カレンダー予定から「次のキャンプカード」を作り、買い物・持ち物・コタ用品・前回反省・リン送信用リストまで一続きにする。

## 実装内容

- 取込元の選択：予約スクショ / PDF / 予約メール / カレンダー予定
- 画像OCR入口：Tesseract.jsをオンデマンド読込
- 予約本文・カレンダー予定の貼付欄
- キャンプ場名、日程、泊数、チェックイン、チェックアウト、住所の候補抽出
- 候補の承認保存
- 次のキャンプカード作成
- 買い物・持ち物・コタ用品・反省候補生成
- リンに送るリスト生成とコピー

## まだMVP完成ではない理由

- OCR精度は実機スクショで検証が必要
- PDF本文の直接抽出は未実装
- Googleカレンダー直連携は未実装
- 候補リストはルールベースで、ギア台帳・過去反省との連携は次ゲート
