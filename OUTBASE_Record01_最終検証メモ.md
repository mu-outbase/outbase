作成日：2026-07-11

## 実装内容

- CLEAN v6の正式承認デザインを維持

- クイック記録を「話す／撮る／動画／場所／ピン／メモ」に統一

- 記録点タップから詳細確認へ遷移
- MAP切替と現在地操作を維持
- GitHub Pages旧キャッシュ対策をRecord-01版へ更新
