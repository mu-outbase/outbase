# OUTBASE USABLE REMAKE

## 目的
「使い方が分からない」「ゴチャしている」「ギア登録/変更・ボックス管理が不明」「散歩の地図と排泄ログの扱いが違う」「カレンダーが見にくい」という指摘を反映した版。

## 主な修正
- ホームは「今日の一手」にして、押す場所を4つに絞った。
- カレンダーは、月移動・日付タップ・ダブルタップ追加・連日予定・繰り返し・日付未設定分離を実装。
- ギアは「一覧 / ボックス / 積込 / 乾燥修理 / 取込」に分離。
- ギア登録・変更・削除、BOX登録・変更、BOX単位積込、車載位置表示を実装。
- 散歩は「通常散歩 / キャンプ場散歩」を分け、地図パネル・GPS点・Google Mapsリンクを実装。
- 排泄ログは表面に出さず、「体調メモ（非公開）」の中だけで記録。
- デザインは暖色背景・紙カード・濃緑アクセント・下ナビ文字のみで37〜40系に寄せた。
- 料理・買い物は献立→材料→買い物統合→LINEコピーを実装。
- バックアップ/復元/ICS書き出しを実装。

## 外部接続
Google Photos / Google Calendar双方向同期 / 天気API本接続 / 家族リアルタイム共有は、この静的PWA単体では本接続していない。
