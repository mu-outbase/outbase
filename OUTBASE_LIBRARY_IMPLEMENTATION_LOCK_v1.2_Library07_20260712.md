# Library07 実装LOCK

## 実機指摘への対応
1. 右側タブが見えない
   - 上段3列、下段4列の固定グリッド。狭幅では2列／3列へ折返し。
2. タブを押すとページが動く
   - 横スクロールを廃止。切替前scrollTopを切替先へ引継ぎ。
3. 全体のズレ
   - ヘッダー、タブ、本文カードの左右余白を16pxへ統一。
4. 文字が大きすぎる
   - 見出しは維持し、本文15px、補足11〜13px中心へ再調整。
5. 1文字記号
   - セット構成の頭文字表示を廃止し、実ギア名ラベルへ変更。

## 維持機能
概要、アイテム、セット、カスタム、在庫、履歴、つながり、車・自転車、ペット、保管場所、取込・出力。
